# 001 - Modelo baseado em PPJ

## Contexto

Precisávamos de uma métrica simples, interpretável e consistente.

## Decisão

Usar PPJ como métrica central.

## Motivo

* Fácil interpretação
* Comparável entre contextos
* Funciona com poucos dados

## Consequências

* Menos sofisticado que xG
* Mais robusto com poucos dados
